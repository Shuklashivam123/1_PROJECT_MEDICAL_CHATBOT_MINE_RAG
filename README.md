Medical Chatbot

Clone my repo :- git clone https://github.com/Shuklashivam123/1_PROJECT_MEDICAL_CHATBOT_MINE_RAG.git

create virtual environment :- conda create -n medicalbot python=3.10

To activate the venv :- conda activate medicalbot

If you want to delete v. environment :- conda remove -n medicalbot --all

To check all the virtual env available in your system :- conda env list

# Install requiremnts by command
